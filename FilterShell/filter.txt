4 loại filter: Authorization, Action, Result, Exception
Chọn loại có ưu tiên cao nhất => Authorization

default order = -1 => để ưu tiên hơn cần set order < -1 (order càng bé thì càng đc ưu tiên)
Thứ tự ưu tiên được xét qua order và scope, trong đó order được ưu tiên trước scope
